This is the primary work that was done for the 1806 and 1828 Noah Websters Dictionaries. The end goal of this was to prepare and clean up the pages to go into headword extraction with minial issues.
Functions such as denoisers, morphology, hughlines, and line detectors were used.
Along with these processes, a model based on Tesseract-OCR was also trained to fixed/lessen rotational issues on pages such that if a page was to appear tilted, the model would revert the tilt to allow for the aformentioned functions to work.
